📘 AI-Powered Study Planner & Focus Tracker

A smart and user-friendly web application designed to help students plan their study schedule, track focus time, and improve productivity with AI-like insights.

🚀 Project Overview

The AI-Powered Study Planner & Focus Tracker is a frontend-based productivity tool built using HTML, CSS, and JavaScript.
It helps users stay organized, maintain focus, and analyze their study habits through an intuitive and modern interface.

This project simulates intelligent behavior using logic-based recommendations—making it feel like a lightweight AI assistant without requiring a backend.

🎯 Key Features
📅 Smart Study Planner
Add subjects, tasks, and deadlines
Automatically generates daily study plans
Stores data using browser local storage
⏱️ Focus Timer (Pomodoro)
25-minute focus sessions with 5-minute breaks
Start, pause, and reset functionality
Helps improve concentration and reduce distractions
📊 Productivity Dashboard
Displays total study hours
Tracks completed tasks
Shows consistency streaks

🧠 AI-like Suggestions
